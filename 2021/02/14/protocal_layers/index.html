网络协议

发表于 2021-02-14 更新于 2021-02-22 分类于 网络


主要就是两种，一种是传统的5层模型，叫做 internet protocol stack，另一种是添加了两层的7层模型，叫 OSI model

Internet Protocol Stack


这里需要留意一下各个数据包的叫法，整体都叫做 packet，从应用层到链路层分别叫：


message: 负责某一种具体服务的协议

segment: 负责线程与线程之间的连接

datagrams: 负责host与host之间的连接

frames: 负责两个实体中介之间的连接，比如路由器与路由器

Seven-layer OSI Model


主要就是在应用层下面加了两层内容


Presentation: 负责加密压缩等任务

Session: 负责双端同步的问题
