不修改数组找出重复的数字

发表于 2021-02-22 分类于 算法

题目描述


长度为 $n+1$ 的数组里所有数字在 $1\sim n$ 的范围内，所以数组中至少有一个数字是重复的。请找出数组中任意一个重复的数字，但不能改变输入的数组。例如，如果输入长度为8的数组 {2,3,5,4,3,2,6,7}，那么对应输出的重复的数字是2或者3

思路


这个题也展现了时间和空间兼顾的问题，对于时间来说，1秒进行百万次的计算是可以承受的，但是对于空间来说，额外建立一个百万级大小的数组是不容易的，所以实际中往往需要先考虑空间的节省，当然时间也不能消耗太多。这个题里面时间复杂度为 $O(nlog(n))$ 空间复杂度为 $O(1)$
